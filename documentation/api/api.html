/kingdom

/kingdom post

post

Create a new kingdom.

post /kingdom

Create a new kingdom.

Secured by JWT

Authorization based on signed token

  • Request
  • Response
  • Security

Body

Media type: application/json

Type: object

Properties
  • width: required (integer - minimum: 1 - maximum: 9)

    The width of the kingdom

    Example:

    8
  • length: required (integer - minimum: 1 - maximum: 9)

    The length of the kingdom

    Example:

    4
  • queen: (object)

    The Queen of kingdom

    • position: required (object)

      Position of the person in the Kingdom

      • x: required (integer)

        Abscissa coordinate

        Example:

        1
      • y: required (integer)

        Ordinate coordnate

        Example:

        2

      Example:

      {
  "x": 1,
  "y": 2
}
    • face: required (one of WEST, NORTH, EAST, SOUTH)

      Direction where a Person is looking on

      Example:

      NORTH

    Example:

    {
  "position": {
    "x": 1,
    "y": 2
  },
  "face": "NORTH"
}
